Trabalhos

Aqui encontra-se algumas de meus trabalhos. Essas ferramentas foram desenvolvidas para atender ou resovler algum problema um demanda dos utilizadores.

Cappellus Web - Loofting

O Cappellus Web é uma ferramenta desenvolvida em parceria com a Infinitysoft voltada para os representates comerciais da Loofting, onde é possivel que seus representates digitem os pedidos em uma interfacil fácil e pratica de usar.

Dessa forma, o setor comercial da Loofting, consegue fazer a avaliação financeira do cliente, e se for aprovado, importar o pedido para seu sistema gerencial, e dar continuidade ao processo, alem de emitir os documentos fiscais

Rastreio - TransApucarana

Com a necessidade de disponibilizar uma interface Web para seus clientes para acompanhar as entregas, foi desenvolvido em parceria com a Spartum Tec um sistema de rastreio, onde alem de acompanhar suas entregas.

Colmeia Verde

O projeto Colmeia verde é uma plataforma inovadora de e-commerce que promove a sustentabilidade ao conectar consumidores com uma variedade de produtos eco-friendly. Com uma curadoria que inclui mais de 5000 itens, o site oferece uma experiência de compra unificada e conveniente, com o bônus de incentivar práticas responsáveis de consumo.

Nossa contribuição foi na implementação de integrações API`s e no desenvolvimento de novas funcionalidades a plataforma

D-Glic - Diário da Glicemia

Gerenciador de Gliceimia É uma uma Aplicativo desenvolvida para auxiliar pessoas que posusem diabetes, e possuem a necssidade de acompanhar a sua glicemia, disponibilizando gráficos com informações da glicemia.

EAD PDV

EAD PDV É uma uma Plataforma Web desenvolvida para aprendizado sobre serviços prestados aos colaboradores e prestadores de serviço da empresa.

SpartumDocs - TransApucarana

O SpartumDocs é um sistema voltado para disponibilizar os Documentos Fiscais da TransApucarana com base nos dados gerados pelo sistema gernecial.

No SpartumDocs, os clientes consegue fazer cotações de transporte, de forma autônomo e prática

Saúde nas Mãos

Saúde nas Mãos foi uma Plataforma Web desenvolvida em um Hackathon, que qualquer pessoa, pode efetuar o cadastro de seus dados médicos, contatos, patologias, remédios, alergias, dentre outros dados médicos

Spartum®Educ

O SpartumEduc foi uma aplicação desenvolvida como forma de patrocinio da Infinitysoft para o XXII SICITE (Seminário de Iniciação Científica e Tecnológica), realizado pela UTFPr - Apucarana, em 2018, onde foi utilizado para avaliar as publicações Científicas.

Spartum®Web

O SpartumWeb é uma ferramenta web, onde é possível gerenciar Clientes, Produtos e Pedidos, sendo integrado com sistemas Gerencias.
